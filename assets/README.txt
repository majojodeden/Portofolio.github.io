================================================================================
ASSETS FOLDER
================================================================================

This folder is for storing icons, logos, and additional assets.

RECOMMENDED ASSETS:
-------------------

1. Icons:
   - favicon.ico (16x16, 32x32, or 48x48px)
   - logo.svg or logo.png (for navbar)

2. Social Media Icons (optional):
   - github-icon.svg
   - instagram-icon.svg
   - linkedin-icon.svg

3. Additional Graphics:
   - Any custom graphics or illustrations

CURRENT STATUS:
---------------
The website currently uses:
- Emoji icons for visual elements
- Google Fonts for typography
- Chart.js for data visualizations

You can replace emoji icons with actual icon files if desired.

FAVICON SETUP:
--------------
To add a favicon:

1. Create or download favicon.ico
2. Place it in the root PORTOFOLIO folder (not in assets)
3. Add to <head> in index.html:
   <link rel="icon" type="image/x-icon" href="favicon.ico">
