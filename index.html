Problem: Finding the optimal route for visiting multiple locations with minimum cost is computationally expensive for large datasets.

Solution: Implemented Branch and Bound algorithm in Java to solve TSP efficiently. Used priority queues for node exploration and pruning strategies to reduce search space. Applied dynamic programming for memoization of subproblems.

Result: Achieved 40% faster computation compared to brute-force approach for 15+ city routes, with route cost visualization.
