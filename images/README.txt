================================================================================
IMAGES FOLDER
================================================================================

This folder is for storing project images and visual assets.

RECOMMENDED IMAGES:
-------------------

1. Project Images (800x600px recommended):
   - data-warehouse.jpg (for SDG 11 project)
   - tsp-optimization.jpg (for Traveling Salesman project)
   - data-logger.jpg (for Microprocessor project)
   - image-recognition.jpg (for Computer Vision project)

2. Profile Photo (optional):
   - profile.jpg or avatar.jpg (400x400px recommended)

3. Background Images (optional):
   - background.jpg (1920x1080px for hero section)

IMAGE FORMATS:
--------------
- JPG/JPEG: For photographs and complex images
- PNG: For images with transparency
- SVG: For scalable icons and logos

OPTIMIZATION:
-------------
- Compress images before uploading
- Use tools like TinyPNG or ImageOptim
- Keep file sizes under 500KB for web performance
